> 파이프를 피하여 오랫동안 비행하면서 살아남는 Flappy Bird 게임입니다.

## Function
- `플레이어` : 이동, 상호작용, 파티클
- `맵`: 파이프 생성
- `아이템` : 속도 증가, 크기 증가, 무적
- `점수` : 시간에 따른 점수, 랭킹
- `설정` : 음량
